rer.block.iframembed
====================

A block which embeds external url inside an iframe

--------

- Domains validation


Installation
------------

Install rer.block.iframembed by adding it to your buildout::

    [buildout]

    ...

    eggs =
        rer.block.iframembed


and then running ``bin/buildout``
